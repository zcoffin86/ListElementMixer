List Element Mixer

Welcome to the List Element Mixer! This tool allows you to create and manage multiple lists of items. You can load lists from text files, save them as presets, and generate random combinations of items from each list. To get started, click "Add List" to create a new list column. Use the "Pick a .txt File" button to load items into the list. You can save your current lists as presets and load them later. The "Generate Output" button will randomly select an item from each list and combine them into a single output string. Happy randomizing!
